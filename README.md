# 🪨 📜 ✂️ Rock Paper Scissors

A modern, sleek, and interactive Rock Paper Scissors game built with Vanilla JavaScript, HTML5, and CSS3. Experience the classic game with a clean UI and smooth interactions.

## ✨ Features

- 🎮 **Interactive Gameplay**: Smooth transitions and responsive design.
- 🤖 **Smart Opponent**: Play against a CPU with randomized logic.
- 📊 **Real-time Scoring**: Track your wins and losses instantly.
- 🎨 **Sleek UI**: Modern glassmorphism-inspired components and clear visual feedback.

---

## 🚀 How to Play

1. **Clone the Repo**:
   ```bash
   git clone https://github.com/BhagyashreeMali/stone_paper_scissors.git
   ```
2. **Launch**: Open `index.html` in your favorite browser.
3. **Fight!**: Click on **Rock**, **Paper**, or **Scissors** to make your move and see if you can beat the computer!

## 🛠️ Built With

- **HTML5**: For semantic structure.
- **CSS3**: For premium styling and animations.
- **JavaScript (ES6+)**: For game logic and DOM manipulation.
