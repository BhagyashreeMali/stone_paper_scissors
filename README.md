## 🎯 About The Project

> _A sleek, interactive, and addictive Rock Paper Scissors game — reimagined for the modern web._

This isn't your ordinary Rock Paper Scissors! Built with clean code and styled for a premium feel, this game packs smooth interactions, instant feedback, and a playful UI that keeps you coming back for more.

## 🎲 How the Game Works

```
┌─────────────────────────────────────────────────┐
│                                                 │
│   🪨 Rock       ──beats──▶   ✂️ Scissors        │
│   📜 Paper      ──beats──▶   🪨 Rock             │
│   ✂️ Scissors   ──beats──▶   📜 Paper            │
│                                                 │
└─────────────────────────────────────────────────┘
```

## 🚀 Quick Start

```bash
# 1️⃣ Clone the repository
git clone https://github.com/BhagyashreeMali/stone_paper_scissors.git

# 2️⃣ Navigate to the project
cd stone_paper_scissors

# 3️⃣ Open in browser
open index.html
```

> 💡 **Tip:** You can also use **Live Server** in VS Code for hot reloading!
